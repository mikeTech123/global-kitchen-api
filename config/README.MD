# The Global Kitchen API

## Tech Stack
- Runtime: Node.js
- Framework: Express.js
- Database: MongoDB
- ODM: Mongoose
- Configuration: dotenv

## Features
- Create recipes
- Get recipes
- Update recipes
- Delete recipes
- Filter by category
